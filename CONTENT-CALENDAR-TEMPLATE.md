## Content Type Descriptions

### Blog Posts (Flagship Content)
**Length:** 1,000-2,500 words
**Format:** Narrative, storytelling, data-driven
**Update Frequency:** Evergreen (update quarterly)
**SEO Focus:** Long-tail keywords, featured snippets
**CTA:** Book consultation, download resource, join community

**Examples:**
- "Why 2025 is Your Year to Transform with AI"
- "The Economics of Custom AI Agents"
- "5 Scaling Mistakes Costing You Thousands"

---

### How-To Guides (Actionable Content)
**Length:** 1,500-3,000 words
**Format:** Step-by-step, process-oriented
**Update Frequency:** Every 6 months or after major changes
**SEO Focus:** Query-based keywords, featured snippets
**CTA:** Download template, schedule training, book call

**Examples:**
- "How to Build Your First AI Automation Workflow"
- "How to Implement Custom AI Agents (In 30 Days)"
- "How to Calculate AI ROI"

---

### Case Studies (Social Proof)
**Length:** 1,500-2,000 words
**Format:** Problem → Solution → Results
**Update Frequency:** New cases quarterly
**SEO Focus:** Brand keywords, specific outcomes
**CTA:** "Get similar results" → Contact/Call

**Examples:**
- "Case Study: $2M Agency → 3x Revenue with AI Agents"
- "Case Study: Automation Saved $400K/Year"
- "Case Study: From Manual to Fully Automated"

---

### Strategic Guides (Deep Content)
**Length:** 3,000-5,000 words
**Format:** Comprehensive, multi-section, reference-worthy
**Update Frequency:** Annually
**SEO Focus:** Competitive, branded terms, thought leadership
**CTA:** Email capture, consultation booking

**Examples:**
- "The Ultimate AI Implementation Playbook"
- "Business Transformation with AI: Complete Guide"
- "AI Strategy for 2025: Executive Perspective"

---

## Publishing Checklist

Use this checklist for every blog post before publishing:

### Content Planning Phase
- [ ] Topic research and keyword analysis complete
- [ ] Outline created and approved
- [ ] Resources/sources identified
- [ ] Data points verified and sourced
- [ ] Case studies/examples collected
- [ ] Target audience defined

### Writing Phase
- [ ] Draft completed (first pass)
- [ ] Self-edit for clarity and flow
- [ ] Check brand voice alignment
- [ ] Verify all claims and statistics
- [ ] Add internal links (3-5 minimum)
- [ ] Create compelling headlines (3+ options)
- [ ] Write meta description (155-160 chars)
- [ ] Draft social media captions (5 variations)

### SEO Optimization Phase
- [ ] Target keyword identified (focus keyword)
- [ ] Secondary keywords identified (3-5)
- [ ] Keyword placement optimized (title, H1, intro, conclusion)
- [ ] Meta title created and optimized
- [ ] Meta description written with CTA
- [ ] Schema markup added (Article schema minimum)
- [ ] Alt text added to all images
- [ ] Internal links added with proper anchor text
- [ ] Heading hierarchy correct (H1, H2, H3)
- [ ] Featured snippet Q&A section created
- [ ] Readability score 70+ (Flesch-Kincaid)
- [ ] Word count optimized (1000+ for blog posts)

### Content Formatting Phase
- [ ] Table of contents added (if 2000+ words)
- [ ] Subheadings are descriptive and keyword-relevant
- [ ] Paragraphs are short (2-4 sentences max)
- [ ] Bullet points for lists (max 5-7 per list)
- [ ] Bold key phrases and statistics
- [ ] Quotes are properly formatted
- [ ] Numbers and dates are current
- [ ] Proper spacing between sections
- [ ] CTA placement optimized (multiple locations)

### Image & Visual Phase
- [ ] Featured image selected/created
- [ ] Image size optimized (1200x630px OG image)
- [ ] All images compressed
- [ ] Alt text descriptive (not keyword-stuffed)
- [ ] Image captions added (if relevant)
- [ ] No broken image links
- [ ] Image file names are descriptive

### Technical Phase
- [ ] No broken internal links (test all)
- [ ] External links open in new tab
- [ ] Mobile responsiveness checked
- [ ] Load time acceptable (<3 seconds)
- [ ] Metadata displays correctly in preview
- [ ] OG image displays properly on social
- [ ] Schema markup validates (schema.org)
- [ ] No spelling/grammar errors (Grammarly)
- [ ] No readability issues (Hemingway Editor)

### Conversion Elements Phase
- [ ] CTA is clear and compelling
- [ ] CTA button is visible and clickable
- [ ] CTA has specific benefit (not "click here")
- [ ] Related posts section included
- [ ] Email capture element present
- [ ] Call-to-action email copy written
- [ ] Landing page/destination confirmed

### Social Preparation Phase
- [ ] 5+ social media caption options written
- [ ] Quote graphics created (2-3)
- [ ] Link preview image confirmed
- [ ] LinkedIn post formatted
- [ ] Twitter/X threads planned
- [ ] TikTok/Shorts script drafted
- [ ] Email subject line (3 options)

### Analytics Setup Phase
- [ ] GA4 conversion goals configured
- [ ] UTM parameters added to CTA links
- [ ] Form tracking set up (if applicable)
- [ ] Page set to track in Google Search Console
- [ ] Anchor links tracked for engagement
- [ ] Heatmap tracking configured

### Final Review Phase
- [ ] Content owner review complete
- [ ] Edit/feedback incorporated
- [ ] One final proofread
- [ ] Link verification complete
- [ ] Schema validation complete
- [ ] Mobile preview checked
- [ ] Desktop preview checked
- [ ] Ready for publish

### Publishing Phase
- [ ] Set publish date and time
- [ ] Add to blog category/tags
- [ ] Set featured image
- [ ] Create excerpt for archives
- [ ] Add to RSS feed
- [ ] Publish post
- [ ] Verify page loads correctly
- [ ] Check metadata displays properly
- [ ] Confirm mobile appearance

### Post-Publishing Phase
- [ ] Add to email newsletter
- [ ] Schedule social media posts (5+ platforms)
- [ ] Share on LinkedIn company page
- [ ] Send email notification to subscribers
- [ ] Monitor comments/engagement
- [ ] Update internal linking strategy
- [ ] Submit to Google Search Console
- [ ] Share in relevant communities
- [ ] Add to content distribution
- [ ] Schedule for social re-sharing (6 months out)
